5 princípios fundamentais da segurança da informação

  1. Integridade;
  2. Confidencialidade;
  3. Disponibilidade;
  4. Autenticidade;
  5. Não-Repúdio;

5 Passos para criar uma senha forte

  1. Use caracteres especiais;
  2. Informações pessoais;
  3. Evite senquênciais;
  4. Nunca repita a mesma em outros apps/sites;
  5. Use a verificação de duas etapas;

Como criar a verificação de duplo fator

  1. Baixe o aplicativo de autenticação no celular;
  2. Cadastre o email no aplicativo;
  3. Certifique-se de ativar a verificação no email;

Os 3 níveis de segurança da informação

  1. Confidencial;

3 Tipos de ataques cibernéticos

  • Ransomware;
  • Phishing;
  • DDoS;

Medidas de segurança para adotar

  • Utilizar a verificação de duplo fator;
  • Usar senhas fortes;
  • Usar sites com encriptação segura;
  • Tomar cuidado com golpes online;

4 Dicas para verificar um email de Phishing

  • Verificar se o email é de procedencia segura;
  • Ver o conteúdo com cautela;
  • Confirmar se o remetente é o correto;
  • Não confiar em emails dúvidosos;

Componentes de um firewall;

  • Filtros externos, usados para proteger os gateways de ataques;
  • Filtro interno, usado para guardar contra as consequências de um gateway comprometido;
  • Gateway, oferece serviço através de proxy;

Consequências de vazamentos de dados de uma empresa

  • Vazamento de senhas dos usuários;
  • Exposição de dados críticos da empresa;
  • Comprometimento do andamento de processos;
  • Criação de uma iamgem negativa para a empresa;
